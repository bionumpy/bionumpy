BnpDataclass
--------------
BioNumPy extends the concept of Python's `dataclass` decorator, by providing a `bnpdataclass`-decorator.
This decorator enables dataclass-like objects with numpy-array-like fields (`EncodedArray`, `EncodedRaggedArray`, `RaggedArray` or pure numpy-arrays). See examples of usage in the documentation below.
